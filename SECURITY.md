# Security Policy

## Supported Versions

To ensure the stability and security of your projects, only the latest major version of the Menu Template receives active updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

As this is a local client-side UI package, the attack surface is minimal. However, if you discover a vulnerability (e.g., an exploit related to JSON deserialization or memory leaks), **please do not report it by creating a public GitHub issue.**

Instead, please report it privately to ensure it can be addressed safely before public disclosure.

1. Send a direct message to: **darpeixlucas@gmail.com**
2. Please include:
   * A clear description of the vulnerability.
   * The Unity version you are using.
   * Step-by-step instructions to reproduce the issue.

You should expect an acknowledgment of your report within 48 hours. Thank you for helping keep this project safe and reliable.
